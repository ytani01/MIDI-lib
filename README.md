# midi-utils

MIDI utilities and test programs

## TL;DR

Install
```bash
$ cd ~
$ python3 -m venv env1
$ cd env1
$ git clone https://github.com/ytani01/midi-utils.git
$ cd midi-utils
$ . ./bin/activate
(env1)$ pip install -r requirements.txt
```

Execute
```bash
$ ./MidiParser.py midi_file

$ ./MidiPaperTape.py midi_file
```
